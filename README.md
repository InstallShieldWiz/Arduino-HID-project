# Arduino-HID-project
Proof of concept for HID USB devices using Arduino IDE to program Digispark HID.

this current version opens up notepad upon plugging in the digispark usb device and injects key strokes to write " You're computer is mine!!!! MUAHAHAHA";

silly I know, just a proof of concept. 

to add a slight change to the program I added an open loop that writes in binary gibberish as a form of proof of concept for ongoing resource drain, memory leak type of attack.
